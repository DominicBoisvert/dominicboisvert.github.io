Enseignement

J’adore l’enseignement. Trouvez ici des ressources que je partage avec mes étudiants et mes collègues.

Ressources bibliographiques

Les ressources bibliographiques des mes cours sont disponibles à partir de Zotero. Elles sont regroupées par sigle de cours ARV3054, SCI6144, etc.

Mon compte est accessible à l’URL https://www.zotero.org/dboisvert/items.

Attention

L’accès à plusieurs ressources bibliographiques est réservé aux abonnés des bibliothèques de l’Université de Montréal. Vous devrez configurer le proxy sur votre ordinateur pour accéder à distance à ces ressources.

Ressources web

Une ressource web, c’est un article de blogue, le site web d’un éditeur de logiciel, le site Github d’un projet. Les ressources web de mes cours sont accessibles par hypothes.is. Tout comme les ressources bibliographiques, les ressources web sont regroupées par sigle de cours. Les sigles sont des tags.

Pour arriver directement sur un ensemble de ressources d’un cours, il est possible de manipuler un URL. Il suffit de modifier l’URL https://hypothes.is/users/dominicboisvert?q=tag%3AARV3054. L’URL précédent pointe vers les ressources taguées avec le sigle ARV3054. Changer le sigle, vous changerez de ressources.
